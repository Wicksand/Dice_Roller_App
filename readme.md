# Dice Roller App

An android app written in Kotlin that simulates rolling the basic dice for RPG style games. 

## Instructions for Build and Use

Steps to build and/or run the software:

1. Ensure that a emulator and android studio are installed. (can also be accomplished by plugging an android device into the computer.)
2. Open the emulator within android studio.
3. press the green play button at the top of Android studio.

Instructions for using the software:

1. Insure that the android application is installed on the device or emulator.
2. Click the app's icon to open and run the program.
3. swipe left and right to select what die you want to roll.
4. press the roll button to generate a number within the possibilities of each dice.
5. The result will display on top of the image of the die and in the saved list of rolls below the roll button.


## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Download android studio
* Download a virtual phone emulator with the 34 API.
* create a Kotlin file with the prebuilt gradle structure.
* Import Jetpack Compose in your main file.
